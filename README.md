# CodeAlpha Stock Portfolio Tracker 

A simple Python script to track your stock investments and calculate your total investment value. The script also saves your portfolio summary to a file.

## Features

- Track multiple stocks like AAPL, TSLA, GOOGL, AMZN.
- Calculate investment based on stock price and quantity.
- Display total investment value.
- Save portfolio summary to a file (`portfolio.txt`).

## How to Use

1. Clone the repository or download the script.
2. Run the Python file:

```bash
python stock_portfolio_tracker.py
